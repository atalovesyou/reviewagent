# ReviewAgent — AI Review Management

**Auto-respond to Google, Yelp, and Facebook reviews in your brand voice.** Sentiment analysis, QR review invites, batch mode.

🔗 **Live:** [https://atalovesyou.github.io/reviewagent/](https://atalovesyou.github.io/reviewagent/)

## Features

- **AI Response Engine** — Type any review, get a brand-appropriate response instantly
- **Batch Mode** — Process multiple reviews at once
- **Sentiment Analysis** — Keyword extraction and trending
- **Voice Builder** — Define your brand voice for consistent responses
- **Request Tracker** — Track review requests sent to customers
- **QR Code Generator** — Dynamic QR codes linking to Google review page
- **Analytics Dashboard** — Sentiment trends, response rate, review velocity
- **100% Client-Side** — No data leaves your browser
